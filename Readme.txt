== Description ==

Плагин WP-Clean удаляет лишние meta-теги и отключает невостребованные функции в CMS WordPress.

WP-Clean удаляет:

* версию WordPress;
* meta-теги, для работы с Windows Live Writer;
* ссылки на соседние статьи;
* раздел "Помощь";
* сообщения в панели управления "Ваш браузер устарел";
* dns-prefetch;
* ссылки из admin bar;
* запись в админ панели, что введенный пароль/логин не верный;
* статические страницы из результатов поиска;
* фразу «Спасибо вам за творчество с WordPress»;
* поле «Сайт» в форме комментариев;
* wp-json;
* oembed;
* embed.

WP Clean отключает:

* стандартные виджеты в панели управления;
* возможность выбора цветовой гаммы в WordPress;
* tracback на себя;
* emoji WordPress (смайлы Emoji);
* автоматическое сохранение записи.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "WP Clean" в директорию `/wp-content/plugins/`;
2. Активируйте "WP Clean" на странице `Плагины` в панели управления.
